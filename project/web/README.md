## 👬 参与贡献

技能图谱使用 [Balsamiq](https://balsamiq.com/products/mockups) 生成，项目源文件在 `/project` 目录。如果你要编辑或修改图谱，请先安装并打开 Balsamiq ，依次点击 **项目** > **导入** > **Mockup JSON** 选项，并将源文件内容粘贴至输入框完成导入。

更新完成后，你需要生成新的源文件和图谱，并提交一个 PR 发起合并请求。
